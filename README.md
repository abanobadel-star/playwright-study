### to run test using docker compose : 
 # docker-compose run --rm playwright-test

### to update docker-compose to get latesr update:
 # docker-compose up 

 ### to close docker-compse to clear it:
 # docker-compose down 

 ### to run test direct from docker file :
# docker run --rm docker-playwright

### to update docker from docker file :
# docker build -t docker-playwright .
